# A. General instructions 
#################################################################################################

This project is designed to be executed with R scripts. 
Execute script files in the order they are listed.

Data sources:

The "observed" precipitation data as collected by DROP members are available in the data section of this repository "https://github.com/MTAV26/thesis/tree/main/Chapter4-4DROP/Data". However, it is important to note that the GPCP dataset, identified as "gpcp_cdr_v23rB1_1981_2021.nc," exceeds the maximum 25 MB limit allowed for downloads on this website.

To access this data, we recommend downloading it through the following web link: https://psl.noaa.gov/data/gridded/data.gpcp.html, or alternatively, establishing direct contact with the responsible author.
